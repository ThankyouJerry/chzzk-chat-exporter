# 설치 가이드

## Chrome에서 확장프로그램 로드하기

### 1단계: 개발자 모드 활성화

1. Chrome 브라우저를 엽니다
2. 주소창에 `chrome://extensions/`를 입력하고 Enter를 누릅니다
3. 우측 상단의 **개발자 모드** 토글을 켭니다

![개발자 모드](https://developer.chrome.com/static/docs/extensions/get-started/tutorial/hello-world/image/extensions-page-e0d64d89a6acf_1920.png)

### 2단계: 확장프로그램 로드

1. **압축해제된 확장 프로그램을 로드합니다** 버튼을 클릭합니다
2. 파일 탐색기에서 `chzzk-chat-exporter` 폴더를 선택합니다
3. **선택** 버튼을 클릭합니다

### 3단계: 확인

확장프로그램이 성공적으로 로드되면:
- 확장프로그램 목록에 "치지직 채팅 내보내기"가 표시됩니다
- 브라우저 툴바에 확장프로그램 아이콘이 나타납니다

## 사용 시작하기

1. 치지직 방송 페이지로 이동합니다
   - 예: `https://chzzk.naver.com/live/...`

2. 확장프로그램 아이콘을 클릭합니다

3. "채팅 수집 시작" 버튼을 클릭하여 채팅 수집을 시작합니다

## 문제 해결

### 확장프로그램이 로드되지 않아요
- 폴더 경로가 올바른지 확인하세요
- `manifest.json` 파일이 폴더 루트에 있는지 확인하세요

### 아이콘이 보이지 않아요
- 확장프로그램 아이콘을 툴바에 고정해보세요
- Chrome 우측 상단의 퍼즐 아이콘을 클릭하여 확장프로그램 목록을 확인하세요

### 치지직 페이지에서 작동하지 않아요
- 페이지를 새로고침해보세요
- 확장프로그램을 비활성화했다가 다시 활성화해보세요

## 업데이트 방법

1. `chrome://extensions/` 페이지로 이동합니다
2. "치지직 채팅 내보내기" 확장프로그램을 찾습니다
3. 새로고침 아이콘(🔄)을 클릭합니다

또는 코드를 수정한 경우:
1. 확장프로그램을 제거합니다
2. 위의 설치 단계를 다시 따릅니다

## 제거 방법

1. `chrome://extensions/` 페이지로 이동합니다
2. "치지직 채팅 내보내기" 확장프로그램을 찾습니다
3. **제거** 버튼을 클릭합니다
4. 확인 대화상자에서 **제거**를 다시 클릭합니다
