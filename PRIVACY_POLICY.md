# 개인정보 처리방침

**최종 수정일**: 2025년 12월 5일

## 개요

치지직 채팅 내보내기(Chzzk Chat Exporter)는 사용자의 개인정보 보호를 최우선으로 생각합니다. 이 문서는 확장프로그램이 어떤 데이터를 수집하고 어떻게 사용하는지 설명합니다.

## 수집하는 정보

### 채팅 데이터
확장프로그램은 사용자가 명시적으로 "채팅 수집 시작" 버튼을 클릭했을 때만 다음 정보를 수집합니다:

- **타임스탬프**: 채팅 메시지가 작성된 시간
- **사용자 ID**: 채팅을 작성한 시청자의 공개 ID
- **메시지 내용**: 채팅 메시지의 텍스트

### 방송 정보
채팅 수집 시 다음 정보를 함께 저장합니다:

- **방송 제목**: 치지직 방송의 제목
- **채널 이름**: 스트리머의 채널 이름
- **방송 URL**: 현재 방송의 URL

## 데이터 저장 방식

- **로컬 저장**: 모든 데이터는 사용자의 브라우저 로컬 스토리지에만 저장됩니다
- **외부 전송 없음**: 수집한 데이터는 어떠한 외부 서버로도 전송되지 않습니다
- **사용자 제어**: 사용자가 직접 데이터 수집을 시작하고 중지할 수 있습니다

## 데이터 사용 목적

수집한 데이터는 오직 다음 목적으로만 사용됩니다:

1. **CSV 내보내기**: 사용자가 요청한 CSV 파일 생성
2. **UI 표시**: 확장프로그램 팝업에서 수집 상태 표시

## 데이터 공유

- ❌ 제3자와 데이터를 공유하지 않습니다
- ❌ 광고 목적으로 데이터를 사용하지 않습니다
- ❌ 분석 도구에 데이터를 전송하지 않습니다

## 사용자 권리

사용자는 다음 권리를 가집니다:

- **수집 제어**: 언제든지 채팅 수집을 시작하거나 중지할 수 있습니다
- **데이터 삭제**: 브라우저의 확장프로그램 데이터를 삭제하여 모든 수집 데이터를 제거할 수 있습니다
- **확장프로그램 제거**: 언제든지 확장프로그램을 제거할 수 있으며, 제거 시 모든 데이터가 삭제됩니다

## 권한 설명

확장프로그램이 요청하는 권한과 그 사용 목적:

### activeTab
- **목적**: 현재 활성화된 치지직 탭에서 채팅 데이터를 읽기 위해 필요
- **사용**: 채팅 메시지 수집에만 사용

### storage
- **목적**: 수집한 채팅 데이터를 로컬에 저장하기 위해 필요
- **사용**: 브라우저 로컬 스토리지에 데이터 저장

### downloads
- **목적**: CSV 파일을 사용자의 컴퓨터에 다운로드하기 위해 필요
- **사용**: CSV 파일 생성 및 다운로드

### host_permissions (chzzk.naver.com)
- **목적**: 치지직 웹사이트에서 확장프로그램이 작동하기 위해 필요
- **사용**: 치지직 페이지의 채팅 요소 접근

## 보안

- **HTTPS**: 모든 통신은 HTTPS를 통해 이루어집니다 (치지직 웹사이트와의 통신)
- **로컬 처리**: 모든 데이터 처리는 사용자의 브라우저 내에서만 이루어집니다
- **오픈소스**: 소스 코드가 공개되어 있어 누구나 검증할 수 있습니다

## 데이터 삭제 방법

### 수집한 채팅 데이터 삭제
1. Chrome 설정 > 개인정보 및 보안 > 사이트 설정
2. 추가 콘텐츠 설정 > 확장 프로그램 데이터
3. "치지직 채팅 내보내기" 찾아서 삭제

### 확장프로그램 완전 제거
1. `chrome://extensions/` 페이지 열기
2. "치지직 채팅 내보내기" 찾기
3. "제거" 버튼 클릭
