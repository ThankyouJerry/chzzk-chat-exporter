# 빠른 시작 가이드 🚀

## 1분 안에 시작하기

### 1️⃣ Chrome에 확장프로그램 설치

1. Chrome 브라우저를 엽니다
2. 주소창에 입력: `chrome://extensions/`
3. 우측 상단 **개발자 모드** 켜기
4. **압축해제된 확장 프로그램을 로드합니다** 클릭
5. `chzzk-chat-exporter` 폴더 선택

### 2️⃣ 치지직 방송 접속

치지직 생방송 또는 다시보기 페이지로 이동:
- 생방송: `https://chzzk.naver.com/live/[채널ID]`
- 다시보기: `https://chzzk.naver.com/video/[비디오ID]`

### 3️⃣ 채팅 수집 시작

1. 브라우저 툴바에서 확장프로그램 아이콘 클릭 💬
2. **채팅 수집 시작** 버튼 클릭
3. 채팅이 실시간으로 수집됩니다!

### 4️⃣ CSV로 내보내기

1. 원하는 만큼 채팅 수집 후 **수집 중지** 클릭
2. **CSV로 내보내기** 버튼 클릭
3. 저장 위치 선택하면 완료! 📥

## 테스트해보기

실제 방송 전에 테스트하고 싶다면:

1. `test-page.html` 파일을 브라우저로 열기
2. **자동 채팅 시작** 버튼 클릭
3. 확장프로그램으로 채팅 수집 테스트

⚠️ **주의**: 테스트 페이지는 실제 치지직 도메인이 아니므로 확장프로그램이 자동으로 작동하지 않을 수 있습니다. 실제 치지직 페이지에서 테스트하세요!

## CSV 파일 열기

### Excel
1. Excel 실행
2. 파일 > 열기
3. CSV 파일 선택
4. 인코딩: UTF-8

### Google Sheets
1. Google Sheets 열기
2. 파일 > 가져오기
3. CSV 파일 업로드

## 문제가 생겼나요?

### 확장프로그램이 보이지 않아요
- Chrome 우측 상단 퍼즐 아이콘(🧩) 클릭
- "치지직 채팅 내보내기" 찾아서 핀 고정

### 채팅이 수집되지 않아요
- 페이지 새로고침 (F5)
- 확장프로그램 비활성화 후 재활성화

### 더 자세한 도움이 필요하다면
- [설치 가이드](INSTALLATION.md) 참고
- [README](README.md) 전체 문서 확인
- GitHub Issues에 문의
