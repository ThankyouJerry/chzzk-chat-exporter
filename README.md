# 치지직 채팅 내보내기 (Chzzk Chat Exporter)

치지직(Chzzk) 방송의 채팅을 CSV 파일로 내보내는 크롬 확장프로그램입니다.

## 주요 기능

- 🎥 **전체 채팅 수집**: 치지직 API를 사용하여 VOD의 모든 채팅을 수집 (영상 재생 불필요!)
- 💾 **CSV 내보내기**: 수집한 채팅을 CSV 파일로 저장
- ⏱️ **타임스탬프 기록**: 각 채팅 메시지의 정확한 시간 기록
- 👤 **사용자 정보**: 채팅을 작성한 시청자의 ID 저장
- 📊 **데이터 분석 가능**: CSV 형식으로 엑셀, 구글 스프레드시트 등에서 분석 가능
- 🚀 **빠른 수집**: 영상을 재생하지 않아도 모든 채팅을 빠르게 가져옴

## 설치 방법

### Chrome Web Store에서 설치 (예정)
추후 Chrome Web Store에 등록될 예정입니다.

### 개발자 모드로 설치

1. 이 저장소를 다운로드하거나 클론합니다
   ```bash
   git clone https://github.com/yourusername/chzzk-chat-exporter.git
   ```

2. Chrome 브라우저에서 `chrome://extensions/` 페이지를 엽니다

3. 우측 상단의 **개발자 모드**를 활성화합니다

4. **압축해제된 확장 프로그램을 로드합니다** 버튼을 클릭합니다

5. 다운로드한 `chzzk-chat-exporter` 폴더를 선택합니다

6. 확장프로그램이 설치되면 브라우저 툴바에 아이콘이 나타납니다

## 사용 방법

1. **치지직 방송 페이지로 이동**
   - 생방송: `https://chzzk.naver.com/live/[채널ID]`
   - 다시보기: `https://chzzk.naver.com/video/[비디오ID]`

2. **확장프로그램 아이콘 클릭**
   - 브라우저 툴바에서 치지직 채팅 내보내기 아이콘을 클릭합니다

3. **채팅 수집 시작**
   - "채팅 수집 시작" 버튼을 클릭합니다
   - 실시간으로 채팅이 수집되며, 수집된 채팅 개수가 표시됩니다

4. **수집 중지**
   - 원하는 만큼 채팅을 수집한 후 "수집 중지" 버튼을 클릭합니다

5. **CSV로 내보내기**
   - "CSV로 내보내기" 버튼을 클릭합니다
   - 저장 위치를 선택하면 CSV 파일이 다운로드됩니다

## CSV 파일 형식

내보낸 CSV 파일은 다음과 같은 형식으로 저장됩니다:

```csv
Timestamp,User ID,Message
2025-12-05T07:30:00.000Z,user123,안녕하세요!
2025-12-05T07:30:05.123Z,user456,ㅋㅋㅋㅋ
2025-12-05T07:30:10.456Z,user789,재밌네요
```

- **Timestamp**: ISO 8601 형식의 타임스탬프
- **User ID**: 채팅을 작성한 시청자의 ID
- **Message**: 채팅 메시지 내용

## 데이터 분석 예시

### Excel에서 열기
1. Excel을 실행합니다
2. "데이터" > "텍스트/CSV에서" 선택
3. 다운로드한 CSV 파일을 선택합니다
4. 인코딩을 "UTF-8"로 설정합니다

### Google Sheets에서 열기
1. Google Sheets를 엽니다
2. "파일" > "가져오기" 선택
3. CSV 파일을 업로드합니다

## 개인정보 보호

이 확장프로그램은:
- ✅ 수집한 데이터를 로컬에만 저장합니다
- ✅ 외부 서버로 데이터를 전송하지 않습니다
- ✅ 사용자의 개인정보를 수집하지 않습니다
- ✅ 오픈소스로 공개되어 있어 코드를 직접 확인할 수 있습니다

자세한 내용은 [개인정보 처리방침](PRIVACY_POLICY.md)을 참조하세요.

## 기술 스택

- **Manifest V3**: 최신 Chrome Extension API 사용
- **Chzzk API**: 치지직 공식 API를 통한 채팅 데이터 수집
- **Chrome Storage API**: 로컬 데이터 저장
- **Chrome Downloads API**: CSV 파일 다운로드

## 문제 해결

### 채팅이 수집되지 않아요
- 페이지를 새로고침한 후 다시 시도해보세요
- 치지직 페이지의 구조가 변경되었을 수 있습니다 (이슈 제보 부탁드립니다)

### CSV 파일이 깨져 보여요
- Excel에서 열 때 인코딩을 "UTF-8"로 설정해주세요
- 메모장이나 텍스트 에디터로 열어보면 정상적으로 보입니다

### 확장프로그램이 작동하지 않아요
- Chrome 브라우저를 최신 버전으로 업데이트해주세요
- 확장프로그램을 비활성화했다가 다시 활성화해보세요
