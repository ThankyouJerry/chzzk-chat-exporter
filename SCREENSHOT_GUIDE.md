# 스크린샷 가이드

Chrome Web Store 제출을 위한 스크린샷 준비 가이드입니다.

## 요구사항

- **최소 개수**: 1개
- **권장 개수**: 3-5개
- **권장 크기**: 1280x800 또는 640x400
- **형식**: PNG 또는 JPEG
- **최대 파일 크기**: 각 5MB

---

## 필수 스크린샷

### 1. 확장프로그램 팝업 - 초기 상태
**파일명**: `screenshot-1-popup-initial.png`

**촬영 방법:**
1. 치지직 생방송 페이지 접속
2. 확장프로그램 아이콘 클릭
3. 팝업이 열린 상태에서 스크린샷

**포함되어야 할 요소:**
- ✅ 확장프로그램 제목 "💬 치지직 채팅 내보내기"
- ✅ 상태 메시지 "✅ 치지직 페이지가 감지되었습니다"
- ✅ 방송 URL 입력 필드
- ✅ "채팅 수집 시작" 버튼
- ✅ 방송 정보 섹션 (제목, 스트리머 이름)

**설명 텍스트:**
```
치지직 방송 페이지에서 확장프로그램을 실행하면 방송 정보가 자동으로 표시됩니다.
```

---

### 2. 채팅 수집 중
**파일명**: `screenshot-2-collecting.png`

**촬영 방법:**
1. "채팅 수집 시작" 버튼 클릭
2. 채팅이 수집되는 동안 (최소 10개 이상 수집 후)
3. 팝업 화면 스크린샷

**포함되어야 할 요소:**
- ✅ 상태 메시지 "🔄 채팅 수집 중..."
- ✅ "수집 중지" 버튼 (활성화 상태)
- ✅ 수집된 채팅 개수 표시 (예: "50")
- ✅ 진행 바 (선택사항)

**설명 텍스트:**
```
실시간으로 채팅을 수집하며, 수집된 채팅 개수가 실시간으로 업데이트됩니다.
```

---

### 3. CSV 내보내기 준비
**파일명**: `screenshot-3-ready-export.png`

**촬영 방법:**
1. 충분한 채팅 수집 후 "수집 중지" 클릭
2. "CSV로 내보내기" 버튼이 활성화된 상태
3. 스크린샷

**포함되어야 할 요소:**
- ✅ 상태 메시지 "✅ 수집이 완료되었습니다"
- ✅ 수집된 채팅 개수 (예: "127")
- ✅ "CSV로 내보내기" 버튼 (활성화, 녹색)
- ✅ 방송 정보

**설명 텍스트:**
```
수집이 완료되면 CSV 파일로 내보낼 수 있습니다. 파일명은 자동으로 생성됩니다.
```

**설명 텍스트:**

### 5. CSV 파일 예시
**파일명**: `screenshot-5-csv-example.png`

**촬영 방법:**
1. 내보낸 CSV 파일을 Excel 또는 Google Sheets로 열기

**포함되어야 할 요소:**


## 스크린샷 촬영 팁

### 1. 고품질 이미지
- 브라우저를 전체 화면으로
- 확장프로그램 팝업이 잘 보이도록
- 불필요한 탭이나 북마크 바는 숨기기

### 2. 일관된 스타일
- 모든 스크린샷을 같은 브라우저에서
- 같은 테마 사용 (라이트 모드 권장)
- 같은 해상도로 촬영

### 3. 실제 데이터 사용
- 테스트 데이터보다는 실제 방송 데이터 사용
- 개인정보가 포함된 경우 모자이크 처리
- 욕설이나 부적절한 내용은 제외

### 4. 주석 추가 (선택사항)
- 중요한 기능에 화살표나 하이라이트 추가
- 간단한 설명 텍스트 오버레이
- 너무 많은 주석은 피하기

## 스크린샷 편집 도구

### macOS
- **기본 스크린샷**: `Cmd + Shift + 4`
- **편집**: Preview 앱
- **주석 추가**: Preview 또는 Skitch

### Windows
- **기본 스크린샷**: `Win + Shift + S`
- **편집**: Paint 또는 Paint 3D
- **주석 추가**: Snipping Tool

### 온라인 도구
- **Canva**: 크기 조정 및 주석 추가
- **Photopea**: 고급 편집
- **TinyPNG**: 파일 크기 압축
