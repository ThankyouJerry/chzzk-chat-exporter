```
🎥 치지직 방송 채팅을 쉽게 수집하고 분석하세요!

치지직 채팅 내보내기는 치지직(Chzzk) 생방송 및 다시보기의 채팅을 실시간으로 수집하여 CSV 파일로 저장할 수 있는 Chrome 확장프로그램입니다.

✨ 주요 기능

💬 실시간 채팅 수집
- 치지직 생방송 또는 다시보기의 채팅을 실시간으로 수집
- 타임스탬프, 사용자 ID, 메시지 내용을 정확하게 기록
- 수집 시작/중지를 자유롭게 제어

📊 CSV 내보내기
- 수집한 채팅을 CSV 파일로 저장
- Excel, Google Sheets 등에서 바로 분석 가능
- UTF-8 인코딩으로 한글 완벽 지원

🎯 직관적인 인터페이스
- 모던하고 사용하기 쉬운 디자인
- 실시간 수집 상태 표시
- 수집된 채팅 개수 실시간 확인

🔒 완벽한 프라이버시
- 모든 데이터는 로컬에만 저장
- 외부 서버로 데이터 전송 없음
- 개인정보 수집 없음
- 오픈소스로 코드 투명성 보장

💡 사용 방법

1. 치지직 방송 페이지로 이동
2. 확장프로그램 아이콘 클릭
3. "채팅 수집 시작" 버튼 클릭
4. 원하는 만큼 채팅 수집
5. "수집 중지" 후 "CSV로 내보내기" 클릭

📋 CSV 파일 형식

내보낸 CSV 파일에는 다음 정보가 포함됩니다:
- Timestamp: 채팅 작성 시간 (ISO 8601 형식)
- User ID: 채팅 작성자의 ID
- Message: 채팅 메시지 내용
```

## 스크린샷 요구사항

최소 1개, 권장 3-5개의 스크린샷이 필요합니다 (1280x800 또는 640x400 권장)

### 필수 스크린샷:

1. **확장프로그램 팝업 메인 화면**
   - 치지직 페이지에서 확장프로그램 아이콘 클릭 시 나타나는 화면
   - "채팅 수집 시작" 버튼이 보이는 상태
   - 방송 정보가 표시된 상태

2. **채팅 수집 중 화면**
   - 채팅이 수집되고 있는 상태
   - 수집된 채팅 개수가 표시된 화면
   - "수집 중지" 버튼이 활성화된 상태

3. **CSV 내보내기 준비 화면**
   - 수집이 완료된 후
   - "CSV로 내보내기" 버튼이 활성화된 상태
   - 수집된 채팅 개수가 표시된 화면

### 추가 스크린샷 (선택사항):

4. **실제 치지직 방송 페이지**
   - 확장프로그램이 작동하는 실제 환경
   - 채팅창이 보이는 화면

5. **내보낸 CSV 파일 예시**
   - Excel 또는 Google Sheets에서 열린 CSV 파일
   - 타임스탬프, 사용자 ID, 메시지가 잘 정리된 모습

## Single Purpose Description

**For Chrome Web Store Review (영어로 작성):**

```
Chzzk Chat Exporter's single purpose is to help users collect and export chat messages from Chzzk live streams. The extension provides a simple interface to start/stop chat collection, view collected messages in real-time, and export them as CSV files for analysis. All functionality is focused on chat data collection and export for Chzzk streaming platform.
```

## Permission Justifications

**For Chrome Web Store Review (영어로 작성):**

### Storage Permission
```
Used to save collected chat messages (timestamp, user ID, message content) and video information (title, channel name) locally on the user's device. No data is transmitted to external servers.
```

### ActiveTab Permission
```
Used to access the URL of the current tab when the user opens the extension popup. This allows the extension to determine if the user is currently on a Chzzk live stream page and provide relevant options.
```

### Downloads Permission
```
Used to export collected chat data as a CSV file to the user's computer when the user clicks the "Export to CSV" button.
```

### Host Permissions (chzzk.naver.com)
```
The extension uses content scripts to observe and collect chat messages from Chzzk live stream pages. This is the core functionality of the extension. The extension only runs on Chzzk pages and captures publicly visible chat messages.
```

## Testing Instructions for Reviewers

**For Chrome Web Store Reviewers (영어로 작성):**

```
1. Install the extension
2. Navigate to a Chzzk live stream page (e.g., https://chzzk.naver.com/live/[channel-id])
3. Click the extension icon in the toolbar
4. Click "채팅 수집 시작" (Start Collection) button
5. Wait for some chat messages to be collected
6. Click "수집 중지" (Stop Collection) button
7. Click "CSV로 내보내기" (Export to CSV) button
8. Verify the CSV file is downloaded with proper format
9. Verify all data is stored locally (check Chrome DevTools → Application → Storage)
```
